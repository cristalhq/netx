Better `net` package for Go.

## Features

* Simple API.
* Dependency-free.
* Clean and tested code.
* Fully compatible with `net` package.

See [GUIDE.md](https://github.com/cristalhq/netx/blob/main/GUIDE.md) for more details.

## Install

Go version 1.17+

```
go get github.com/cristalhq/netx
```
